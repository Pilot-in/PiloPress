# Sync layout

- Add folder `pilopress/layouts/your-layout/` with your layout files in it (PHP, JS, CSS, JSON).
- Go to `Pilo'Press > Layouts > Sync available` and sync your layout field group.

**NB:** JS and CSS files are not require.
